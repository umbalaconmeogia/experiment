Development environment setup

Setup PostgreSQL

  1. Install Postgre SQL
    You can download and install the latest version of Postgres SQL from http://www.enterprisedb.com/products/pgdownload.do#windows.
    To run the below command at command prompt, you should set the path of PostgreSQL\bin to the PATH environment variable of the Windows.
    You need php-pdo-pgsql to be installed into PHP, too.
  2. Create the database yii2rbac for the current system by running the following command at the command prompt.
    createdb -U postgres -E UTF8 yii2rbac
    On linux, you need run the command as user postgres sudo -u postgres createdb -U postgres -E UTF8 yii2rbac

    To delete the database

    dropdb -U postgres yii2rbac
    On linux, you need run the command as user postgres sudo -u postgres dropdb -U postgres yii2rbac
  3. Create the user yii2rbac
    1. Run PostgreSQL client at the command prompt
      psql -U postgres
    2. Run the following PostgreSQL commands to create the user and grant privileges
      CREATE USER yii2rbac WITH PASSWORD 'yii2rbac';
GRANT ALL PRIVILEGES ON DATABASE yii2rbac TO yii2rbac;
  4. To import the data schema (don't do this. Use migration instead)
    
psql -U yii2rbac -d yii2rbac -f schema.pgsql.sql
